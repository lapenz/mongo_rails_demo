# Blog
## 1. Go to root folder and Run bundle install at the first time:

```bash
bundle install
```

## 2. Yarn:

```bash
yarn install
```

## 3. Run the App:

```bash
rails s
```

## Details about the APP:

I have used Cancancan for authorization.
The internationalization gets the Language preference from the Browser.

## Tests

```bash
rspec
```
